# Crafting Interpreters

This repository contains my implementation while working through [Crafting Interpreters](https://craftinginterpreters.com/) by Robert Nystrom.

## Project Structure

The book implements two interpreters:
- **jlox**: A tree-walk interpreter in Java (Part II)
- **clox**: A bytecode virtual machine in C (Part III)
